TOWERS OF HANOI – JavaScript Console Game

Rules
Move one disc at a time
Only move the top disc from a peg
No larger disc on a smaller disc
Win when all discs are stacked correctly on peg 2 or peg 3

Tech Used
JavaScript (no libraries)
Arrays & 2D arrays
Objects, loops, and conditionals
Git & GitHub

How to Run
Open the file in a browser
Open the Developer Console
Print the board:
board.printBoard()
Move discs:
board.moveDisc(1, 3)


Peg numbers must be 1–3.

Features
Validates moves
Detects the winning condition automatically
Prints a congratulations message
Resets the game after winning
